# 3d-bee-project-

3D Interactive Bee Website

Tech Stack: HTML, CSS, JavaScript, <model-viewer>, Intersection Observer API

Developed an immersive 3D interactive website featuring a real-time animated bee model using <model-viewer>
Implemented scroll-based 3D animations by dynamically controlling camera orbit and object positioning
Designed a modern UI with glassmorphism effects, smooth transitions, and responsive layout
Built section-based animation system using Intersection Observer for reveal-on-scroll effects
Created interactive navigation system with active link tracking and smooth scrolling
Developed a functional contact form UI with loading and success states using JavaScript
Optimized performance with efficient DOM manipulation and event handling techniques
Ensured cross-device responsiveness with custom media queries
